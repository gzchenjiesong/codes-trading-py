## 弹窗设计

### 添加标的弹窗

**布局：**
- 基础参数：代码、名称、市场、模式
- 基础参数：基础资金、单网格限制、三网步进%
- **高级参数（折叠）：** 使用 `<details>` 标签实现可折叠区域
  - 最大回撤%、触发加点
  - 最小买入%、清仓步进%、底部买入%
  - 利率%、利息间隔、分红再投资

**交互：**
- 默认折叠高级参数，减少认知负担
- 点击"高级参数（可选）"展开/折叠
- 展开后显示所有高级配置项

**样式：**
```css
details {
  margin-bottom: 12px;
}
summary {
  cursor: pointer;
  font-size: 13px;
  color: #07c160;
  margin-bottom: 8px;
}
```

## 网格表格设计

**表格列：**
- 类型（小网/中网/大网）- 彩色标签
- 档位（L1, L2, ...）
- 买入价、买入数
- 卖出价、卖出数
- 止盈利、清仓利
- 触发状态（左侧红色边框高亮）

**响应式：**
- 小屏：横向滚动（`overflow-x: auto`）
- 大屏：字体稍大（13px）
