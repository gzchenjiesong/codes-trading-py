# 首页 / 导航模块 — index.md

## 功能概述

应用主页，提供模块切换导航（标的/市场/资讯三个Tab）和标的列表管理。

## 页面结构

### 顶部导航栏
- 标题："📊 网格交易"
- 用户信息（登录后显示昵称，点击可退出登录）

### 主体区域

**未登录：** 显示登录引导
- Logo + 应用名称
- 微信登录按钮
- 开发者快速登录

**已登录：** 标的卡片列表
- 每个卡片：名称、代码、当前价格、模式标签(M1/M2/M3)、基础资金、三网步进
- 点击卡片 → 进入详情（网格分析 + 交易记录）

### 底部 Tab 栏（固定定位）

| Tab | 图标 | 路由 |
|-----|------|------|
| 资讯 | 📰 | /news |
| 市场 | 📈 | /market |
| 标的 | 📋 | / （当前高亮） |
| 添加 | ➕ | 弹窗 |

## 标的详情页

点击标的卡片后展开：
- 返回按钮
- 价格更新输入框
- 网格分析表（档位、买入/卖出价、数量、触发状态）
- 持仓分析（净持仓、成本价、浮盈亏、收益率）
- 盈亏总览（总投入、市值、总盈亏）
- 交易记录列表 + 添加交易

## 交互流程

```
页面加载 → GET /api/auth/me
  → 未登录？显示登录页
  → 已登录？GET /api/stocks → 渲染卡片列表

点击卡片 → GET /api/stocks/:id → 渲染详情
添加交易 → POST /api/stocks/:id/trades → 刷新详情
```
