# 市场模块 — market.md

## 功能概述

提供全球核心指数实时行情、行业/概念板块排行、用户自选关注板块。

## 页面路由

`GET /market` → `market.html`

## 页面结构

### 核心 7 指数

| 指数 | 代码 | 市场 | 数据源 |
|------|------|------|--------|
| 中证 A500 | 000510 | A 股大盘基准 | 腾讯财经 |
| 中证 2000 | 932000 | A 股小微盘 | 腾讯财经 |
| 科创 50 | 000688 | 硬科技 | 腾讯财经 |
| 恒生指数 | HSI | 港股 | 腾讯财经 |
| 恒生科技 | HSTECH | 港股科技 | 腾讯财经 |
| 标普 500 | .INX | 美股大盘 | 新浪财经 |
| 纳斯达克 | .IXIC | 美股科技 | 新浪财经 |

每个指数显示为信息卡：名称、实时价格、涨跌幅、迷你 K 线图（Canvas 绘制）。
点击卡片展开 120 日蜡烛图弹窗。

### 板块排行

- Tab 切换：行业板块 / 概念板块
- 涨跌幅 TOP 10（涨/跌分列）
- 每个板块显示：名称、涨跌幅、涨跌家数

### 我的板块

用户自定义关注板块列表：
- ＋ 添加按钮 → 弹窗搜索全部板块
- 每个板块卡片显示实时涨跌幅
- × 删除已关注板块
- 需要登录态

## API 端点

| 方法 | 路径 | 说明 | 鉴权 |
|------|------|------|------|
| GET | `/api/market/indices` | 7 指数实时行情 | 无 |
| GET | `/api/market/index-kline/:code` | 指数日 K 线 | 无 |
| GET | `/api/market/sectors` | 全部板块列表 | 无 |
| GET | `/api/market/sectors/ranking` | 板块涨跌排行 | 无 |
| GET | `/api/user/sectors` | 用户关注板块 | ✅ |
| POST | `/api/user/sectors` | 添加关注板块 | ✅ |
| DELETE | `/api/user/sectors` | 移除关注板块 | ✅ |

## 数据源

| 数据 | 来源 | API |
|------|------|-----|
| A 股指数 | 腾讯财经 | qt.gtimg.cn |
| 港股指数 | 腾讯财经 | qt.gtimg.cn |
| 美股指数 | 新浪财经 | hq.sinajs.cn |
| 指数 K 线 | 腾讯财经 CDN | web.ifzq.cn |
| 行业板块 | 东方财富 | push2.eastmoney.com |
| 概念板块 | 东方财富 | push2.eastmoney.com |

## 设计要点

- 7 指数覆盖 A 股大盘/小微盘/科技 + 港股大盘/科技 + 美股大盘/科技
- 板块不搞 Top N 排序，用户自选关注为主
- 迷你 K 线用 Canvas 绘制，不依赖第三方图表库
- 点击指数卡片弹出 K 线详情
